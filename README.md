Material didático aberto sobre **Controladores Lógicos Programáveis
(CLP)** desenvolvido para estudantes do **Curso Técnico em
Eletrotécnica**.

Este projeto utiliza **Quarto** para gerar uma **apostila digital
navegável** publicada via **GitHub Pages**.

A versão publicada do material pode ser acessada em:

**https://haulisson.github.io/clp/**

------------------------------------------------------------------------

# Sobre o Projeto

Esta apostila foi desenvolvida com os seguintes objetivos:

-   apoiar o ensino de **Automação Industrial**
-   integrar **teoria e prática**
-   apresentar **lógica de comandos elétricos aplicada a CLPs**
-   facilitar o aprendizado progressivo de **Ladder e GRAFCET**
-   permitir expansão contínua do material didático

O material foi pensado para cursos técnicos e disciplinas de:

-   Automação Industrial
-   Comandos Elétricos
-   CLP
-   Sistemas Digitais aplicados à Automação

------------------------------------------------------------------------

# Estrutura do Projeto

O site é construído com **Quarto** e organizado por capítulos.

    Apostila-CLP
    │
    ├── index.qmd
    ├── _quarto.yml
    ├── styles.css
    ├── README.md
    │
    └── content
        └── cap01-introducao
             └── index.qmd

Cada capítulo da apostila possui sua própria pasta dentro de `content/`.

------------------------------------------------------------------------

# Estrutura da Apostila

## Parte I --- Fundamentos de CLP

-   Introdução aos Controladores Lógicos Programáveis
-   Evolução dos sistemas de controle industrial
-   Arquitetura de CLPs
-   Endereçamento de entradas e saídas
-   Linguagem Ladder
-   Temporizadores e contadores
-   Intertravamentos industriais

## Parte II --- Automação Sequencial

-   Introdução ao **GRAFCET**
-   Estruturas sequenciais
-   Sistemas automáticos industriais
-   Integração entre GRAFCET e Ladder

# Simulação de Sistemas

Durante o desenvolvimento das aulas podem ser utilizados simuladores de
CLP como:

https://plciosim.com/

Essas ferramentas permitem testar programas Ladder em ambientes
simulados antes da implementação em equipamentos reais.
